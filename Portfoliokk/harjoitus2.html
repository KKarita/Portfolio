Harjoitus 2 - Puhelinluettelo

Toteutettiin puhelinluettelo, jossa käyttäjä voi lisätä nimiä ja puhelinnumeroita taulukkoon. 

        
        var input = require("readline-sync");

        let phonebook = [];

        function findPhoneNum(phonebook, name) {
            for (const person of phonebook) {
                if (person.name === name) {
                    return person.phone;
                }
            }
            return "Haettua numeroa ei löytynyt.";
        }

        while (true) {
            let name = input.question("Anna nimi: ");
            if (name === "") break;
            let phone = input.question("Anna puhelinnumero: ");
            phonebook.push({ name: name, phone: phone });
        }

Miten tein?

Käytin JavaScriptin taulukkoa ja objektirakennetta tallentaakseni puhelinluettelon tiedot. Lisäksi loin funktion, joka hakee tietyn henkilön puhelinnumeron nimen perusteella.

Miksi tein?

Tarkoituksena oli harjoitella JavaScriptin perusrakenteita, kuten taulukkoja, objekteja ja silmukoita.

Mitä opin?

Harjoitus opetti, kuinka listoja käsitellään JavaScriptissä ja kuinka käyttäjän syötteitä voidaan tallentaa ja hakea taulukosta.

Miten onnistuin?

Tehtävä onnistui hyvin, mutta sitä voisi jatkaa niin, että ohjelma myös tallentaisi numerot sovelluksen sulkemisen jälkeen.
