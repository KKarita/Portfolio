            document.getElementById("nimi").value = nimi;
            document.getElementById("puhelin").value = puhelin;
            editingId = id;
            document.getElementById("saveChangesBtn").style.display = "block";
            document.getElementById("submitBtn").style.display = "none";
        }

Miten tehtiin?

Lisättiin "Muokkaa" -painike, joka täyttää lomakkeen olemassa olevilla tiedoilla ja sallii niiden päivittämisen. Muokkaa -painike tulee näkyviin vasta, kun käyttäjä on valinnut muokattavan numeron. Tämä selkeyttää käyttöliittymää.

Mitä opittiin?

HTTP-metodin `PUT` käyttäminen tietojen päivittämiseen. Opittiin kuinka server asennetaan ja käynnistetään
