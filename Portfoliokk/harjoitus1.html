Harjoitus 1 - Palindromitesti

Tässä tehtävässä testataan, onko annettu sana palindromi.

        
        function palindrome(word) {
            let reversed = "";
            for (let i = word.length - 1; i >= 0; i--) {
                reversed += word[i];
            }
            return word === reversed;
        }

Funktiota testattiin seuraavasti:

        
            const testWord = "nenonen";
            console.log(`Onko "${testWord}" palindromi?`, palindrome(testWord));
            
            const testWordTwo = "kissa";
            console.log(`Onko "${testWordTwo}" palindromi?`, palindrome(testWordTwo));

Mitä tehtiin?

Toteutettiin JavaScript-sovellus, joka testaa, onko annettu sana palindromi.

Miten tehtiin?

Käytin for-silmukkaa merkkijonon kääntämiseen. Tehtävässä ei saanut käyttää valmiita funktioita.

Miksi tehtiin?

Haluttiin harjoitella merkkijonojen käsittelyä ja for-silmukan käyttöä.

Mitä opittiin?

Opittiin kuinka merkkijono voidaan kääntää ilman valmiita metodeja.

Mitä jäi oppimatta?

Voisi tutkia lisää JavaScriptin kehittyneempiä merkkijonokäsittelymenetelmiä.

Miten onnistuin?

Tehtävä onnistui ihan hyvin.
